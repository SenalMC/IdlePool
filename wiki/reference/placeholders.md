# 变量速查

## ActionBar 与 BossBar

消息中的可用变量：

```text
{pool}
{session_time}
{total_time}
{next_reward}
{earned}
{players}
{event}
{multiplier}
```

ActionBar 还支持 IA 图标令牌：

```text
:clock:
:gift:
:coin:
```

## 命令奖励

```text
{player}
{uuid}
{pool}
```
