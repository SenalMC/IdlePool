# 五分钟快速配置

## 1. 创建奖励方案

执行：

```text
/afkpool admin rewards
```

点击“创建奖励方案”，在聊天输入小写 ID，例如 `spawn_rewards`。

你可以：

- 手持物品后点击“添加手持物品”
- 输入 `provider:item_id` 添加模板物品
- 添加 Vault 货币
- 添加控制台命令
- 设置概率、权重、触发方式和时间

## 2. 创建挂机池

站在区域内执行：

```text
/afkpool admin
```

点击“创建挂机池”，输入 ID，例如 `spawn`。

## 3. 配置区域

在区域的两个对角分别点击“设置区域点 1”和“设置区域点 2”。两个点会自动组成长方体。

## 4. 设置玩法

在挂机池详情中设置：

- 奖励周期，例如 `10m`
- 进度保留，例如 `7d`
- 最大挂机人数，`0` 表示不限
- 使用权限
- 奖励方案

## 5. 启用并测试

点击状态按钮启用挂机池。使用普通玩家账号进入区域，确认界面弹出；点击开始后观察 ActionBar，并临时把周期设短测试奖励。

最后执行：

```text
/afkpool doctor
```
